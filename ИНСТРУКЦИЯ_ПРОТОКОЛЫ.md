# Инструкция по размещению протоколов на сайте

## ✅ ЧТО СДЕЛАНО:

1. **Создана папка** `public/documents/protocols/` для хранения PDF файлов
2. **Добавлена секция на главной странице** с 4 карточками протоколов (сразу после hero-секции)
3. **Добавлены стили** для красивого отображения карточек с анимацией при наведении

## 📋 ЧТО НУЖНО СДЕЛАТЬ:

### Скопируйте PDF файлы в папку `public/documents/protocols/`:

Переименуйте и скопируйте файлы следующим образом:

1. **Математика**: 
   - Скопируйте файл протокола по математике
   - Переименуйте в: `Отборочный_этап_Протокол_Математика.pdf`

2. **Физика**: 
   - Из: `Публикация_Отборочный_этап_Общий_протокол_Физика.pdf`
   - В: `Отборочный_этап_Протокол_Физика.pdf`

3. **Химия**: 
   - Из: `Публикация_Отборочный_этап_Общий_протокол_Химия.pdf`
   - В: `Отборочный_этап_Протокол_Химия.pdf`

4. **Биология**: 
   - Скопируйте файл протокола по биологии
   - Переименуйте в: `Отборочный_этап_Протокол_Биология.pdf`

### Команды для копирования (PowerShell):

```powershell
# Перейдите в папку проекта
cd D:\site_apo

# Скопируйте файлы (замените пути на актуальные):
Copy-Item "ПУТЬ_К_ФАЙЛУ_МАТЕМАТИКА.pdf" -Destination "public\documents\protocols\Отборочный_этап_Протокол_Математика.pdf"
Copy-Item "ПУТЬ_К_ФАЙЛУ_ФИЗИКА.pdf" -Destination "public\documents\protocols\Отборочный_этап_Протокол_Физика.pdf"
Copy-Item "ПУТЬ_К_ФАЙЛУ_ХИМИЯ.pdf" -Destination "public\documents\protocols\Отборочный_этап_Протокол_Химия.pdf"
Copy-Item "ПУТЬ_К_ФАЙЛУ_БИОЛОГИЯ.pdf" -Destination "public\documents\protocols\Отборочный_этап_Протокол_Биология.pdf"
```

## 🎨 КАК ЭТО ВЫГЛЯДИТ:

На главной странице появится новая секция с 4 карточками:
- **Математика** (синяя, иконка калькулятора)
- **Физика** (голубая, иконка молнии)
- **Химия** (зелёная, иконка капли)
- **Биология** (жёлтая, иконка цветка)

Каждая карточка:
- Имеет красивую анимацию при наведении
- Содержит кнопку "Скачать протокол"
- Открывает PDF в новой вкладке

## 🚀 ЗАПУСК:

После копирования файлов просто запустите сайт:

```bash
npm run dev
```

Протоколы будут доступны на главной странице сразу после hero-секции!
