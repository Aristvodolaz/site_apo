# 🔧 Настройка Yandex SMTP для рассылки

## ⚠️ Важно: Правильные настройки для Yandex

Yandex SMTP имеет особенности, которые нужно учитывать:

### Вариант 1: Порт 465 (SSL) - Рекомендуется

```env
SMTP_HOST=smtp.yandex.ru
SMTP_PORT=465
SMTP_SECURE=true
SMTP_USER=ваш_email@yandex.ru
SMTP_PASSWORD=ваш_пароль_приложения
SMTP_FROM="Арктическая олимпиада <ваш_email@yandex.ru>"
```

### Вариант 2: Порт 587 (STARTTLS)

Если порт 465 не работает, попробуйте:

```env
SMTP_HOST=smtp.yandex.ru
SMTP_PORT=587
SMTP_SECURE=false
SMTP_USER=ваш_email@yandex.ru
SMTP_PASSWORD=ваш_пароль_приложения
SMTP_FROM="Арктическая олимпиада <ваш_email@yandex.ru>"
```

## 🔑 Как получить пароль приложения Yandex

1. Зайдите в [Настройки безопасности Yandex](https://id.yandex.ru/security)
2. Найдите раздел **"Пароли приложений"**
3. Нажмите **"Создать новый пароль"**
4. Выберите **"Почта"** или **"Другое"**
5. Дайте название (например, "Arctic Olympiad Email")
6. **Скопируйте пароль** (он показывается только один раз!)

⚠️ **ВАЖНО:** Используйте **пароль приложения**, а НЕ обычный пароль от аккаунта!

## 🐛 Решение проблем

### Ошибка "Greeting never received"

**Причина:** Неправильные настройки порта или secure соединения

**Решение:**
1. Попробуйте порт **587** вместо 465:
   ```
   SMTP_PORT=587
   SMTP_SECURE=false
   ```

2. Или попробуйте порт **465** с secure:
   ```
   SMTP_PORT=465
   SMTP_SECURE=true
   ```

3. Проверьте, что используете **пароль приложения**, а не обычный пароль

### Ошибка "Invalid login"

**Причина:** Неправильный пароль или email

**Решение:**
1. Убедитесь, что используете **пароль приложения**
2. Проверьте, что email указан полностью (например, `user@yandex.ru`)
3. Убедитесь, что пароль скопирован без пробелов

### Ошибка "Connection timeout"

**Причина:** Проблемы с сетью или файрволом

**Решение:**
1. Проверьте подключение к интернету
2. Убедитесь, что порты 465 или 587 не заблокированы
3. Попробуйте другой порт (587 вместо 465 или наоборот)

## ✅ Проверка настроек

После настройки `.env` файла запустите тест:

```bash
npm run send-test-email
```

Если тест проходит успешно, значит настройки правильные!

## 📝 Пример правильного .env файла

```env
# Yandex SMTP настройки
SMTP_HOST=smtp.yandex.ru
SMTP_PORT=465
SMTP_SECURE=true
SMTP_USER=arctic@yandex.ru
SMTP_PASSWORD=abcdefghijklmnop
SMTP_FROM="Арктическая олимпиада <arctic@yandex.ru>"
```

---

**Если проблемы остаются:**
1. Проверьте, что аккаунт Yandex активен
2. Убедитесь, что двухфакторная аутентификация включена (для создания пароля приложения)
3. Попробуйте создать новый пароль приложения
